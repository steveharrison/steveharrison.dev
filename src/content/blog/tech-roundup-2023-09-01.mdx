A few weeks back I mentioned [Nova](https://nova.app/), a new, native macOS code editor. I have been using it every day for work and I'm loving it! Interestingly enough, it seems there is a whole bunch of native macOS code editors out now:

- [https://www.chimehq.com/](https://www.chimehq.com/)
- [https://www.codeedit.app/](https://www.codeedit.app/)
- [https://coderunnerapp.com/](https://coderunnerapp.com/)
- [https://coteditor.com/](https://coteditor.com/)

My favourite is CodeEdit, which has heavily copied the Xcode and Apple Developer design:
